Nomoottori is a graphics/game engine written in C and OpenGL. I started the project in November 2023 as a challenge and for fun. It uses software rendering and raycasting for rendering. (i.e. looks like Duke Nukem 3D).

I don't have a downloadable copy of this project and I'm not planning on doing one for quite some time.
