KodeOS is a 32bit operating system written in x86 assembly. I started the project on sixth grade for fun (it was not). The OS features a simple file system, a BASIC (the language) compiler some applications like text editor, music listening software and stuff like that. It was actually usable for everything else but internet since it doesn't have internet access or a web browser.
